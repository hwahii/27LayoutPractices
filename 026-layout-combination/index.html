前端技術 Front-end

前端作為與使用者接觸的最前沿，如何設計、實作前端界面是相當重要的。

若將前端構成的元素簡單切開，可以分成三大類別，各類別都各自有著其精深之處！

HTML5

超文本標記語言是一種用於建立網頁的標準標記語言。HTML5是最新的修訂版本，由全球資訊網協會於2014年10月完成標準制定。目標是取代1999年所制定的HTML4.01和XHTML1.0標準，以期能在網際網路應用迅速發展的時候，使網路標準達到符合當代的網路需求。

CSS3

階層式樣式表（Cascading Style Sheets）是一種用來為結構化文件（如HTML文件或XML應用）添加樣式（字型、間距和顏色等）的電腦語言，由W3C定義和維護。目前最新版本是CSS2.1，為W3C的推薦標準。CSS3現在已被大部分現代瀏覽器支援，而下一版的CSS4仍在開發中。

JavaScript

JavaScript是一種進階的、直譯的程式語言。JavaScript是一門基於原型、函式先行的語言，是一門多範式的語言，它支援物件導向編程，指令式程式設計，以及函式語言程式設計。它提供語法來操控文字、陣列、日期以及正規表示式等，不支援I/O，比如網路、儲存和圖形等，但這些都可以由它的宿主環境提供支援。

後端 Back-end

後端綜合了所有使用者碰觸不到的元素，包括商業邏輯、資料庫等等。

API Server

作為資料的接口，與前端客戶端互動。

資料庫

儲存所有資料的地方，常見有SQL或noSQL兩個大分類，各有優缺點，依照使用需求應用於專案中。

DevOps

DevOps是Development和Operations的組合詞，將開發到營運的流程做有效的整合，近年特化成為一專門的領域。
